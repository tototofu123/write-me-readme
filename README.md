# Write Me README

> [!NOTE]
> Auto-generate beautiful README files for your GitHub repositories

## Overview

This repository contains the **auto-readme** skill for OpenCode. It automatically generates professional README files with rich formatting including tables, emojis, badges, and code blocks.

## Features

- :sparkles: **Auto-Detection** - Automatically detect repository structure
- :art: **Rich Formatting** - Tables, emojis, code blocks, badges
- :rocket: **Multiple Styles** - Simple to detailed output options
- :wrench: **Customizable** - Adapt output to your preferences
- :white_check_mark: **CI Ready** - Generate badges for build status

## Output Sections

| Section | Description |
|---------|-------------|
| Header | Title with emoji, version badge, description |
| Quick Stats | Table with repo info |
| Features | Bulleted list with emojis |
| Commands | Table format for CLI |
| Installation | Code blocks |
| File Structure | Tree visualization |
| Scripts | Table format |
| Badges | License, version, CI status |

## Usage

```
User: "update the README for this repo"
→ Generate rich README with all formatting options

User: "make it simple"
→ Generate basic README without fancy formatting
```

## Rich Formatting Elements

- **Tables**: Feature comparison, command reference
- **Emojis**: :rocket:, :book:, :wrench:, :sparkles:
- **Badges**: Version, license, CI status
- **Code blocks**: bash, javascript, json
- **Callouts**: > [!NOTE], > [!TIP], > [!WARNING]
